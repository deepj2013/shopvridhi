# E-commerce + SaaS Unified Operating Model

## Core Model
ShopVridhi runs as one platform with dual engine:
- Engine 1: Shop Operations (POS, stock, purchase, ledger)
- Engine 2: Commerce Growth (catalog, online orders, delivery, commission)

## Shared Data Backbone
- inventory and pricing feed both in-store and online
- customer profile unifies offline and online behavior
- order-to-bill conversion preserves tax and ledger integrity

## Revenue Model Layers
- SaaS subscription plans
- WhatsApp and premium add-ons
- marketplace commission on qualified transactions
- value services (analytics, automation, premium support)

## Universal Admin Role
Single admin plane controls both engines with scoped access and audit trail.

## Operational Priorities
1. never break billing speed
2. preserve financial and stock correctness
3. keep catalog and order quality high
4. optimize merchant growth loops
