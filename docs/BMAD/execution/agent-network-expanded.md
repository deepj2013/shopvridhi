# Expanded BMAD Agent Network (v2)

## Why Expanded
The initial BMAD setup is strong for baseline delivery, but ShopVridhi needs deeper specialization to beat competitors with execution speed and production quality.

## Agent Layers

### Layer A: Core Delivery
- BA Agent
- Technical Head Agent
- DB Head Agent
- UI/UX Head Agent
- QA Head Agent
- DevOps/SRE Agent
- Program Manager Agent

### Layer B: Competitive Differentiation
- Forms Optimization Agent
- E-commerce CommerceOps Agent
- Growth Marketplace Agent
- Data Analytics Agent

### Layer C: SaaS Governance and Scale
- Admin Control Tower Agent
- RBAC and Identity Agent
- Pricing and SaaS Billing Agent
- Security and Compliance Agent
- Product Strategy Agent

## Cross-Agent Contracts
- BA -> Forms/UIUX/DB: final field-level requirements
- RBAC -> Tech/UIUX/QA: permission behavior by action
- Admin Control Tower -> Pricing/Security/Data: policy enforcement and visibility
- CommerceOps -> Growth/DB/QA: order SLA and conversion quality
- PM -> all: integrated milestone governance

## Decision Hierarchy
1. Shared Goal
2. PM integrated delivery plan
3. Tech and DB feasibility decisions
4. UX and Forms execution constraints
5. Growth/Commerce optimization
