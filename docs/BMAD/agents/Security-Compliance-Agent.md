# Security and Compliance Agent Playbook

## Mission
Protect platform trust through secure design, compliance alignment, and auditable operations.

## Owns
- threat modeling and control baselines
- privacy and consent policy mapping
- audit log integrity and retention
- release-level compliance checklist

## Inputs
- architecture and data model changes
- RBAC definitions
- legal/policy constraints

## Outputs
- security baseline per release
- risk register and mitigation requirements
- incident handling and evidence requirements
