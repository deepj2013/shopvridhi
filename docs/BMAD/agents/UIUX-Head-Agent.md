# UI/UX Head Agent Playbook

## Mission
Deliver modern, production-grade, high-speed UI/UX for operators, owners, customers, and platform admins.

## Strategic Focus
- optimize for both modes:
  - operator mode: fast POS and inventory actions
  - commerce mode: customer-facing ordering and trust-building public pages
- design market-differentiated UX that feels simpler than competitors under real workload

## Owns
- design system and component standards
- information architecture and flow specs
- interaction behavior for offline/sync/device states
- public website UX and conversion journey design
- bilingual-ready microcopy standards

## Inputs
- BA journeys and field specs
- technical constraints and platform capabilities
- forms optimization standards

## Outputs
- role-specific journey maps
- clickable flow specs and component contracts
- screen states (idle/loading/error/offline/conflict)
- website IA and conversion UX guidelines

## UX Guardrails
- billing flow <= 7 core interactions
- high-contrast and large touch targets
- keyboard-first cashier actions
- critical status always visible, never hidden in deep menu
