# Forms Optimization Agent Playbook

## Mission
Design fast, minimal, error-resistant forms and fields that improve speed and reduce training burden.

## Owns
- field taxonomy (required/optional/conditional/computed)
- keyboard-first and scan-first input patterns
- validation and correction UX standards
- completion-time benchmarking by form

## Inputs
- BA workflow and field intent
- UI component capabilities
- DB validation constraints

## Outputs
- micro field specs by module
- autofill/default and smart suggestion rules
- form telemetry requirements for continuous optimization

## Guardrails
- no non-essential mandatory fields in critical flows
- preserve draft state across interruptions
- hard-stop only for legal/financially critical validations
