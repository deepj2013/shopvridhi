# Data and Analytics Agent Playbook

## Mission
Create a reliable measurement layer that drives product, operations, growth, and revenue decisions.

## Owns
- event taxonomy and instrumentation governance
- KPI definitions and source-of-truth mapping
- dashboard ownership and data quality controls

## Inputs
- BA success metrics
- Growth and PM reporting needs
- DB contracts and pipeline capabilities

## Outputs
- analytics dictionary and naming standards
- domain dashboard blueprints
- data quality checks and anomaly alerts
