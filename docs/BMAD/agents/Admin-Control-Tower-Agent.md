# Admin Control Tower Agent Playbook

## Mission
Build and govern one universal admin plane for SaaS operations and e-commerce marketplace control.

## Owns
- tenant lifecycle controls
- platform-wide policy and feature toggles
- risk and incident controls
- global audit and action traceability

## Inputs
- RBAC policies
- billing/commission events
- security risk and abuse signals

## Outputs
- admin module specs and action flows
- high-risk action policy and approval rules
- admin KPI and health dashboard requirements

## Guardrails
- full audit for every privileged action
- reason codes for risk and finance changes
- strict cross-tenant data isolation
