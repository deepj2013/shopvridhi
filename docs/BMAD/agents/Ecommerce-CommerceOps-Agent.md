# E-commerce CommerceOps Agent Playbook

## Mission
Ensure high-quality order operations from discovery to fulfillment to conversion into POS billing.

## Owns
- catalog readiness and listing quality controls
- order routing, exception, and SLA policies
- pickup/delivery operations design
- order-to-bill conversion reliability

## Inputs
- catalog and stock signals
- marketplace demand and rejection metrics
- delivery partner performance data

## Outputs
- operational SOPs for commerce flows
- exception handling playbooks
- SLA dashboard and remediation loops
