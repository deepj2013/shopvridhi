# ShopVridhi Workspace Structure

This repo is now separated by purpose:

- `docs/` -> product, planning, architecture, BMAD execution
- `Code/` -> all implementation code

## Code Layout

- `Code/SaaS/public-website/web` -> SaaS web app (public pages + app shell)
- `Code/SaaS/working/api` -> API server (auth, RBAC, inventory, e-commerce, admin)
- `Code/SaaS/working/mobile` -> mobile workspace
- `Code/SaaS/working/windows` -> windows workspace
- `Code/Ecommerce/public-website` -> ecommerce public website area
- `Code/Ecommerce/working` -> ecommerce working product area
- `Code/shared/saas-ecommerce` -> shared contracts, models, utilities

## Docs Layout

- `docs/BMAD` -> agent and execution documentation
- `docs/project-management` -> TODOs, trackers, execution control
- `docs/architecture` -> technical architecture docs
- `docs/product` -> product docs and legacy reference files
